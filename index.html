Keep it Simple Stupid (KISS) Grid

A functional grid system in only 9 lines of CSS. Compatible with IE 7 and above!

Download Github

Try it out

Add any number of child divs to a .grid and they will automatically form equal sized columns:

Column

How to use it

Either download the CSS file or copy the CSS code below to create the foundation of the grid:

KISS Grid CSS


  .grid {
    display: table;
    table-layout: fixed;
    width: 100%;
  }
  .grid > div {
    display: table-cell;
    vertical-align: top;
  }

Example HTML


  <div class="grid">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
    <div>Four</div>
    <div>Five</div>
    <div>Six</div>
  </div>

Which should result in this (styled for easy viewing):

One
Two
Three
Four
Five
Six

And that's it! The .grid element will act as a full-width row, and its children divs become flexible, class-agnostic, equal-width columns.

Vertical Alignment

Create your own class for the column divs with the attribute vertical-align: top|middle|bottom; to align your content. By default the columns are aligned to the top.

Example CSS


  .custom-class {
    vertical-align: middle;
  }

Example HTML


  <div class="grid">
    <div class="custom-class">I'm content that is vertically aligned!</div>
    <div class="custom-class">Even when there is more content in a column it's still aligned!</p></div>
    <div class="custom-class">Also when there is less!</p></div>
  </div>
