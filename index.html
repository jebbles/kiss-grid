Keep it Simple Stupid (KISS) Grid

A functional grid system in only 8 lines of CSS. Compatible with IE 7 and above!

Download Github

How to use it

Either download the CSS file or copy the CSS code below to create the foundation of the grid:


.grid {
display: table;
table-layout: fixed;
width: 100%;
}
.grid > div {
display: table-cell;
}

Now to create the grid, try the following HTML markup:


<div class="grid">
<div>Col One</div>
<div>Col Two</div>
<div>Col Three</div>
<div>Col Four</div>
</div>

Which should result in this (styled for easy viewing):

Col One
Col Two
Col Three
Col Four

And that's it! The .grid element will act as a full-width row, and its 4 children divs become flexible, class-agnostic, equal-width columns.
